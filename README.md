# Internet Connection Logger for Raspberry Pi


## Installation

```console
git clone https://github.com/aarongoa/network-connection-monitor
npm install
```

## Usage

```console
npm start
```


## Configuration

Create file `downtime.txt` in root directory alongside index.js

```console
touch downtime.txt
```
